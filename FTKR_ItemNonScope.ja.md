# [FTKR_ItemNonScope](FTKR_ItemNonScope.js) プラグイン

アイテムが範囲なしの場合の仕様を変えるプラグインです。

## 概要

本プラグインを有効にすると、アイテムの基本設定「範囲」が「なし」の場合、アクター選択画面を表示せずにパーティーの全メンバーを対象にしてアイテムを使用できます。

ただし、メモ欄に以下のタグを設定したアイテムは本機能を無効にします。
```
<INS 無効>
<INS Disable>
```
